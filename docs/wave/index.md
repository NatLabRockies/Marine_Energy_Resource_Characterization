# Wave Energy Datasets

High-resolution wave hindcast datasets covering the U.S. Exclusive Economic Zone, generated using WaveWatch III and SWAN models.

## Available Datasets

| Dataset | Coverage | Duration | Resolution |
| ------- | -------- | -------- | ---------- |
| [Ocean Surface Wave Hindcast](hindcast/index.md) | U.S. EEZ (West Coast, Atlantic, Hawaii) | 32 years (1979-2010) | 3-hourly |

## Ocean Surface Wave Hindcast

The [High Resolution Ocean Surface Wave Hindcast](hindcast/index.md) is the highest resolution publicly available long-term wave hindcast dataset covering the entire U.S. Exclusive Economic Zone. It provides wave parameters including significant wave height, peak period, wave power, and directional characteristics.
